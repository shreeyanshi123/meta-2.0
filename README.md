# AI Agent Oversight Tribunal

**Theme:** Multi-Agent Interactions (#1)

**Pitch:** A hackathon-grade OpenEnv environment where a Judge LLM is trained to oversee 4 worker agents, detecting and classifying injected failures like hallucinations, collusion, manipulation, and silence. Evaluated by a multi-component rubric-based reward system.
